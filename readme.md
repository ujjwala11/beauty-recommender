# AI Beauty Intelligence Platform

##  Hybrid Recommendation System for Personalized Beauty Discovery

https://beauty-recommender-efdi3c5kqnehprnmkiuizr.streamlit.app/
 
## Overview

An AI-powered beauty recommendation platform that combines **Machine Learning, NLP, Recommendation Systems, and Explainable AI** to provide personalized skincare and beauty product recommendations.

The system learns from **product information, ingredients, customer reviews, and user preferences** to recommend suitable products with clear explanations behind every suggestion.

---

## Features

### 🧴 Product Intelligence
- Ingredient and product analysis
- Feature engineering from metadata and reviews
- Beauty-focused product understanding

### 🤖 Machine Learning Engine
- Rating prediction (Regression)
- Recommendation prediction (Classification)
- Model evaluation and SHAP-based explainability

### 🎯 Recommendation System
- Popularity-based recommendations
- Content-based filtering (TF-IDF + similarity)
- Collaborative filtering
- Hybrid recommendation engine

### 💡 Explainable AI
Recommendations are explained using:
- Ingredient similarity
- Product characteristics
- User preferences
- Review signals
